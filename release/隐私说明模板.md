# 做饭小程序隐私说明模板

以下是一版适合当前 MVP 的简化模板，你后面可以按你自己的主体信息替换。

## 用户隐私保护指引示例
本指引由“【你的主体名称】”为“做饭小程序”制定，用于说明当前版本如何处理用户信息。

### 1. 当前版本处理的信息范围
当前版本主要用于用户在小程序内进行菜品选择、查看热量、金额和食材汇总。

当前版本不要求用户注册账号，也不提供登录、支付、定位、相机、录音、相册上传、通讯录、手机号获取等功能。

当前版本不主动收集用户的昵称、头像、手机号、位置信息、相册、通讯录、身份证件等个人信息。

### 2. 本地使用说明
用户在小程序中的点餐选择，仅用于当前页面内展示所选菜品、热量、金额和食材汇总。

如当前版本未接入云端账户、后端数据库或第三方统计服务，则相关选择信息不会用于跨设备同步，也不会用于对外共享。

### 3. 第三方服务说明
当前版本未接入第三方插件或第三方个人信息处理服务。

如后续版本新增云同步、登录、图片上传、推荐算法或第三方服务，将在新版本中同步更新隐私说明。

### 4. 用户权益
如你对本小程序的信息处理方式有疑问，或希望反馈问题、提出删除建议，可通过以下方式联系开发者。
