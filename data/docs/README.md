# Runtime Data

当前只保留小程序运行和 Excel 查看会用到的数据表，不保留整理过程表。

## 保留的运行时表

- `meal-periods.js`: 早 / 中 / 晚
- `regions.js`: 地域
- `cuisines.js`: 菜系
- `dish-categories.js`: 菜品大类
- `dish-subcategories.js`: 最小子类
- `dietary-types.js`: 肉 / 海鲜 / 素
- `ingredients.js`: 食材词典
- `dishes.js`: 菜品主表
- `dish-metrics.js`: 热量 / 价格 / 份量
- `dish-meal-periods.js`: 菜和餐次关联
- `dish-ingredients.js`: 菜和食材关联

## 页面当前会用到的信息

- 餐次
- 菜名
- 热量
- 价格
- 食材
- 菜系 / 地域 / 分类
